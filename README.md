# UUID-Generator-Intellij-plugin
Random UUID generator plugin for intellij
The UUID will be save to clipboard.

In the Main menu After Tools will be new menu "Tanel Plugins"

MainMenu
 |-> File
 |-> Edit
 |-> Navigate
 |-> Code
 |-> Analyze
 |-> Refactor
 |-> Build
 |-> Run
 |-> Tools
 |-> Tanel Plugins
      |-> Generate UUID to clipboard
 |-> VCS
 |-> Window
 |-> Help
